# Skella Website

This repository contains the codebase for the Skella website. Skella is a web application built for [describe purpose here]. 

## Getting Started

To get started with the Skella website locally, follow these instructions:

1. Clone this repository to your local machine.
2. Install dependencies using npm:
   `npm install`

## Usage

### Development Mode

To run the website locally in development mode, use the following command:
  `npm run dev`
This will start a development server.

### Deployment

To build and deploy the website to GitHub Pages, use the following command:
  `npm run deploy`

This command will build the website and publish it to the `gh-pages` branch of this repository.
